## Hey, I'm Zak Laughton

I'm a full-stack developer and frontend architect based in Austin, TX. I build efficient, maintainable applications and write about the tools and techniques that make development faster and less painful.

My focus is on **JavaScript**, **React**, **GraphQL**, **TypeScript**, and testing, with a particular interest in developer productivity and workflows that scale.

## What I write about

The posts on this site are the things I wish existed when I was learning them. If I spent too long figuring something out, I write it up so the next person doesn't have to.

Topics include: practical Git techniques, testing strategies with Jest, VS Code workflows, and general web development patterns.
